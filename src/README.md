# Monitoramento climático
Construimos um pipeline de dados completo para ingestão, processamento, armazenamento e análise de dados climáticos de múltiplas cidades usando uma API pública.

## Como abrir o repositório na sua máquina
1. Clonar o repositório:

    ```sh
    git clone https://github.com/nicolashrq/Projeto-Monitoramento-Climatico-Bigdata.git
    ```

2. Entrar na pasta do projeto:

   ```sh
   cd Projeto-Monitoramento-Climatico-Bigdata
   ```
## Como Instalar/Configurar:

1.Clone o repositório
    ```sh
    git clone https://github.com/nicolashrq/Projeto-Monitoramento-Climatico-Bigdata.git
    ```

2.Entre na pasta src
    ```sh
    cd src
    ```

3.Instale o Pandas
    ```sh
    pip install pandas
    ```

4.Execute o pipeline
    ```sh
    python run_pipeline.py
    ```
